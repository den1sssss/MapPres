### Installation & Run
Clone the repo and change to the project root directory:
```
git clone https://github.com/0x0000dead/technohack-gas.git
cd technohack-gas
```

Install requirements:
```
pip install -r requirements.txt
```

And run:
```
streamlit run StreamLit.py
```
